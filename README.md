# Calculatrice PySide6

Une application de calculatrice simple développée avec PySide6.

## Installation

1. Installez les dépendances :
```bash
pip install -r requirements.txt
```

## Utilisation

Lancez l'application avec :
```bash
python calculatrice.py
```

## Fonctionnalités

- Saisie de deux nombres (a et b)
- Calcul automatique de la somme
- Bouton "Calculer" pour effectuer l'addition
- Bouton "Annuler" pour vider tous les champs
- Gestion des erreurs pour les saisies invalides

## Interface

L'application présente une interface simple avec :
- Un titre "Calculatrice" en rouge
- Trois champs de saisie étiquetés "a=", "b=" et "Somme=" en bleu
- Le champ "Somme=" est en lecture seule et affiche le résultat
- Deux boutons stylisés "Calculer" et "Annuler"
